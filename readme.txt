== Installation ==

**From the WordPress Admin:**
1. Navigate to **Plugins > Add New**.
2. Search for **"Just Custom Tabs"**.
3. Click **Install Now**, then **Activate**.

**Manual Installation:**
1. Download the plugin ZIP file from the [WordPress Repository](https://wordpress.org/plugins/just-custom-tabs) or [GitHub Repository](https://github.com/JustThereUK/Just-Custom-Tabs).
2. Upload the extracted folder to **wp-content/plugins/**.
3. Go to **Plugins > Installed Plugins**, find "Just Custom Tabs", and click **Activate**.

== Frequently Asked Questions ==

= Can I add tabs to individual products? =
Yes! You can create custom tabs for individual products directly from the product edit screen.

= Can I create tabs that appear on all products? =
Yes. Global tabs can be managed from the plugin settings and will display on every product unless overridden.

= What content can I add inside a tab? =
Anything you can add to the WordPress editor â€“ text, images, HTML, shortcodes, video embeds, etc.

= Can I reorder the tabs? =
Absolutely. Use the drag-and-drop interface to sort tabs in the order you prefer.

= Will it work with my theme? =
Yes! Just Custom Tabs uses standard WooCommerce hooks and works with most themes out of the box.

= Does it slow down my website? =
No. The plugin is lightweight and optimised for performance.
